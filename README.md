# news

> 一个基于Vue2.0的新闻APP
## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
采用了Vue-router做路由 webpack-simple打包发布，采用axios获取数据
